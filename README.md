# Exam portfolio
Two assessment questions from the MSc Bioinformatics programme, answering questions requiring the bulk transcriptomics analysis (R) and supervised classification of biological data (python).
Rendered outputs are included (HTML, .docx); source code and data are provided for inspection and reproduction.

## AML_transcriptomics
Differential expression analysis on a public microarray dataset (GSE9476) inspecting AML vs. healthy bone marrow & peripheral blood. 

Performed using R 4.5.0 with tidyverse, VennDiagram, and limma (Bioconductor)

Data source: Stirewalt DL et al. Identification of genes with abnormal expression changes in acute myeloid leukemia. Genes Chromosomes Cancer 2008; 47(1):8-20. DOI: 10.1002/gcc.20500

## cachexia_ML
Binary classification of cancer-associated skeletal muscle wasting from urinary metabolite profiles.

Performed using python 3.12 with pandas, numpy, scikit-learn, seaborn, and matplotlib.

Data source: Eisner R et al. Learning to predict cancer-associated skeletal muscle wasting from 1H-NMR profiles of urinary metabolites. Metabolomics 2011; 7:25-34. DOI: 10.1007/s11306-010-0232-9
